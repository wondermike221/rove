Node types exercised

  • Action nodes — callback fires immediately
  • Text input — single-line, persisted
  • Textarea — multi-line (Enter = newline, Ctrl+Enter = accept)
  • Checkbox — boolean toggle
  • Select — single choice dropdown
  • Select-multiple — multi-choice list
  • Virtual node — async load with loading overlay
  • Directory nodes — nested navigation
  • Meta directory — built-in settings (theme, mode, shortcuts)

Features

  • Shadow DOM — zero style leakage
  • Fuzzy search across all leaf nodes
  • Palette pinned top or bottom
  • Dir view drag + resize + persist position
  • Light / dark / system theme
  • Focus-scoped shortcuts
  • Multi-instance isolation
  • localStorage persistence
